Formatting a Table

HTML Structure

Tags

  • table: table
  • table: table title
  • thead: Table header
  • tbody: Table content
  • tr: Row of the table
  • th: Title cell of the table
  • td: Cell of a table

Attributes

  • colspan: defines the number of collumns to be merged
  • rowspan: defines the number of rows to be merged
  • scope: defines the td cells to which the th cell is linked (row
    / col)
